As letras "W" e "Y" são especiais porque, às vezes, fazem o som de uma vogal (como "A", "E", "I", "O", "U"). Outras vezes, fazem o som de uma consoante (como "B", "C", "D"...).

Exemplos:

- O "Y" em "yoga" faz o som da vogal "I".
- O "W" em "Walter" faz o som da consoante "V".

Assim, essas letras são como camaleões: mudam de acordo com a palavra em que estão!




Nossos Jogos


Jogo da Memória

Este é um divertido jogo que você precisa encontrar os pares de imagens iguais.
Ao final do jogo, você pode clicar nas sílabas para aprender os sons delas.


Jogo: Qual é o Animal?

Neste jogo, o objetivo é acertar os nomes dos animais antes que o tempo acabe.
Cada vez que você acerta, você ganha um ponto.


Jogo das Sílabas

Este é um jogo que você pode praticar sua habilidade de formar palavras juntando sílabas corretamente.
A cada acerto, você ganha uma estrela.
